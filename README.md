# nadir-point
Computing the nadir point for multiobjective discrete optimization problems

### Abstract
The nadir point is constructed from the worst objective values over the efficient set of a multiobjective optimization problem. This algorithm computes nadir values for multiobjective discrete optimization (MODO) with *p* objective functions. The proposed algorithm is based on an exhaustive search of the *(p-2)*-dimensional space for each component of the nadir point using epsilon-constracint method.
